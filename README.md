Construction Project Management Dashboard
Project Overview
This project is an interactive Construction Project Management Dashboard developed using Microsoft Excel.
The dashboard helps monitor project budget, actual expenditure, project progress, task status, and cost variance through KPI cards, Pivot Tables, and charts.
