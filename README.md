# 🏗️ Construction Project Management Dashboard

## 📌 Project Overview

This project is an interactive Construction Project Management Dashboard developed using Microsoft Excel.

The dashboard helps track project budget, actual expenditure, project completion status, delayed activities, and cost variance using KPI cards, Pivot Tables, and charts.

It provides a simple and effective way to monitor construction project performance and support project management decision-making.

---

## 🚀 Features

- Budget Tracking
- Actual Cost Monitoring
- Cost Variance Analysis
- Project Completion Tracking
- Delayed Task Monitoring
- Interactive Pivot Tables
- Project Status Distribution Analysis
- Planned vs Actual Cost Comparison
- Professional Dashboard Design
